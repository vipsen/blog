## はじめに

このブログは **Gemini Deep Research API** を活用して、AI 分野の最新動向を自動で調査・記事化し、GitHub Pages で公開しています。

## 仕組み

1. Google Apps Script のタイマートリガーが週3回（月・水・金）起動
2. Gemini Flash がその週の注目トピックを選定
3. Deep Research エージェントが 80〜160 回の Web 検索で深掘り調査
4. 調査結果を Gemini Flash がブログ記事に変換
5. GitHub Contents API で自動コミット → GitHub Pages にデプロイ
